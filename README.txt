DriveSim
=========================

This a project for generating a simulation of a vehicle driving around streets.
Input is a network of points connected into ways.
Output a random path taken within a network.

-------
input
-------
a OpenStreetMap XML (OSM)
http://wiki.openstreetmap.org/wiki/OSM_XML

-------
Output
-------
KML
http://www.opengeospatial.org/standards/kml
